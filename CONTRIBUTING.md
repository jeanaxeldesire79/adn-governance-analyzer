# Contributing

We welcome contributions! Here's how you can help:

## Development

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes to `src/index.html`
4. Test locally with `python3 -m http.server 8000`
5. Commit with clear messages
6. Push and open a Pull Request

## Guidelines

- Keep the app as a single-file HTML (no build step)
- Use vanilla JS — avoid external dependencies
- CSS custom properties for theming
- Handle API errors gracefully (offline mode)
- Test on desktop, tablet, and mobile viewports

## Reporting Issues

Open an issue with:
- Description of the problem
- Steps to reproduce
- Browser/device info
- Screenshots if applicable
